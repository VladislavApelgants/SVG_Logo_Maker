## Installation

npm install

## Usage

npm start

    - Enter up to three characters: qwe
    - Enter the text color (color keyword or hexadecimal number)
    - Choose a shape: "circle", "triangle", "square"
    - Enter the shape color (color keyword or hexadecimal number)
